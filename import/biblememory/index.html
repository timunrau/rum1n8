Import from BibleMemory.com

Use this guide to export your BibleMemory.com verses into a CSV file, then bring that CSV into Ruminate.

The exporter runs inside your signed-in BibleMemory browser session. Ruminate does not receive your BibleMemory credentials or verse data until you choose to import the downloaded CSV.

01

Install the exporter

Drag this button to your browser bookmarks bar. It creates a bookmarklet you can run from BibleMemory.com.

Export to Ruminate

If dragging does not work

  1. Create a new bookmark named Export to Ruminate.
  2. Copy the code below.
  3. Paste it into the bookmark's URL or Address field.
Loading bookmarklet...

02

Sign in to BibleMemory.com

Open BibleMemory.com in the same browser and sign in. The exporter can only read the verses that your browser session can already access.

03

Run it from My Verses

Go to My Verses in BibleMemory.com, then click your Export to Ruminate bookmark. A Ruminate export panel should appear on the page while it gathers your verses and collections.

04

Download the CSV

When the export finishes, click Download CSV. Keep the file somewhere easy to find, such as Downloads or your desktop.

Review timing is imported as closely as BibleMemory exposes it. Some review progress may be approximate because BibleMemory does not expose every detail of its scheduler.

05

Import into Ruminate

Open Ruminate, go to Collections, tap +, choose Import CSV, and select the file you downloaded.
